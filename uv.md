**uv** is an extremely fast, all-in-one Python package and project manager written in **Rust** by **Astral** 

1. Installation https://docs.astral.sh/uv/getting-started/installation/
2. Create virtual environment `uv venv`
3. Initialize project `uv init`
4. install packajes from requirements.txt `uv add -r requirements.txt`
5. installing individual library `uv add library_name`
6. Install all packages from pyproject.toml `uv sync`
